## Sobre o livro

O *Opus de Emendatione Temporum* (1583) é a obra fundadora da cronologia histórica como disciplina moderna. Joseph Scaliger introduziu o **Período Juliano** — sistema cronológico ainda usado por astrônomos hoje — e organizou pela primeira vez os calendários antigos (egípcio, babilônico, judaico, helenístico, romano) num quadro coerente.

Em 443 anos de existência, **nunca foi integralmente traduzido para nenhuma língua vernácula**.
