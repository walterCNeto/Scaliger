# data/

Esta pasta abriga os datasets gerados pelo pipeline:

- `events_normalized.jsonl` — 1.643 eventos astronômicos extraídos
- `audit_eclipses.jsonl` — auditoria por eclipse contra NASA
- `audit_final.jsonl` — auditoria completa com todos os tipos de eventos
- `audit_dataset.csv` — versão tabular para análise
- `audit_summary.json` — sumário agregado
- `nasa_solar_eclipses.csv` — Five Millennium Canon (Espenak/Meeus)
- `nasa_lunar_eclipses.csv` — idem

Os datasets são regenerados pelo pipeline. Não edite manualmente.
