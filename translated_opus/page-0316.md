# Página 316

## Português

DA EMENDAÇÃO DOS TEMPOS, LIVRO IV.

Esta página contém os fastos romanos (calendário) para os meses de abril, maio, junho, julho, agosto e setembro, dispostos em seis colunas, com letras nundinais (A–H), indicações dos dias (K = Kalendae, NON = Nonae, EID = Idus), caráter do dia (F = fastus, N = nefastus, C = comitialis, NP = nefastus publicus, EN = endotercisus), e notas sobre festas, jogos e eventos históricos. Ver seção tables para reprodução completa.

## English

ON THE EMENDATION OF TIMES, BOOK IV.

This page contains the Roman fasti (calendar) for the months of April, May, June, July, August, and September, arranged in six columns, with nundinal letters (A–H), day markers (K = Kalendae, NON = Nonae, EID = Idus), day character (F = fastus, N = nefastus, C = comitialis, NP = nefastus publicus, EN = endotercisus), and notes on feasts, games, and historical events. See the tables section for full reproduction.

### Flags de incerteza

- Coluna de setembro, linha 'B': registro ambíguo — aparece vazio ou com entrada 'LVD. ROM. IN CIRC.' sem marcador de dia claro.
- Coluna de julho: abreviações MERK. (= Mercatus) e identificação da letra nundinal final 'C' para o dia 30 repetem-se; grafia e ordem das letras seguem o original.
- Algumas letras nundinais iniciais (A, B, C, D) aparecem em corpo maior ao lado esquerdo de grupos de dias, parecendo indicar divisão nundinal; foram registradas mas podem ter função estrutural distinta.


### Notas do tradutor

Página integralmente ocupada pelos fastos romanos dispostos em seis colunas (abril a setembro). Abreviações técnicas: K = Kalendae; NON = Nonae; EID = Idus; F = fastus; N = nefastus; C = comitialis; NP = nefastus publicus (feriado); EN = endotercisus; FP = feriae publicae; LVDI = ludi; MERK = Mercatus; CAR. MONET = Carmentalis/Monetae; MART. CAR. MONET. = Marti (aedes), ou festividade monetária; MATR. MAG. = Matri Magnae; CER. = Cereri; PAR. = Parilia; VIN. = Vinalia; ROB. = Robigalia; FLOR = Floralia; LEM. = Lemuria; AGON = Agonalia; TVB. = Tubilustrium; Q. REX C. F = Quando Rex Comitiavit Fas; MART. CAR. MONET = dedicatio Iunonis Monetae; VEST. = Vestalia; MATR. = Matralia; Q. ST. DF = Quando Stercus Delatum Fas; POPLIF. = Poplifugia; APOLLIN = Apollinares; LVCAR. = Lucaria; NEPT. = Neptunalia; FVRR. = Furrinalia; PORT. = Portunalia; CONS. = Consualia; VOLC. = Volcanalia; OPIC. = Opiconsivia; VOLT. = Volturnalia. As notas históricas registram vitórias de César na Hispânia (victoria Hispaniensis; vitória em Híspalis/Sevilha) e a dedicação da Ara Victoriae na Cúria, além do natalício de Augusto.
